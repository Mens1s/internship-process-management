# University Internship Management System

This project is designed to **streamline the internship application and evaluation process** for universities. It allows **students to submit internship applications** and enables **university administrators to review and manage these applications efficiently**.

## Features
- Students can **apply for internships** through the system.
- University staff can **evaluate and approve/reject applications**.
- A structured **internship management workflow** for both students and administrators.
- **User-friendly interface** developed with modern web technologies.
